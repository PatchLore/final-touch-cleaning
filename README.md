# Final Touch Cleaning

End of tenancy cleaning service website built with Next.js 14, Tailwind CSS, and GSAP animations.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui
- GSAP + ScrollTrigger
- Lucide React

## Getting Started
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Online quote request form
- Service breakdown by room type
- Animated landing page
- Mobile responsive

## Deployment

Deployed via [Vercel](https://vercel.com).
